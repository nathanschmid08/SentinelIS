# SentinelIS CSS Style Guide

## Colors

- **Primary colors:** Deep navy blues for text and accents, e.g., `#0f172a`, `#1e293b`, `#2563eb`

- **Secondary colors:** Emerald green (`#10b981`), amber (`#f59e0b`), red (`#ef4444`) for status indicators

- **Background colors:** Light slate grays(`#f8fafc`, `#f1f5f9`, `#e2e8f0`) for surfaces, cards, and panels

- **Borders:** Subtle gray borders (`#e2e8f0`, `#f1f5f9`)

- **Text colors:** Dark text (`#0f172a`), muted text (`#64748b`)

## Typography

- **Font family:** Modern sans-serif, e.g., `Inter`, system fonts

- **Font sizes:** Base `14px`, headings scale from `16px → 32px`

- **Font weights:** Regular (`400`), medium (`500`), semi-bold (`600`), bold (`700`)

- **Line height:** `1.5` for body text

- **Letter spacing:** Small adjustments for headings (e.g., `-0.3px`) and uppercase labels

## Layout & Spacing

- **Sidebar:** Fixed vertical panel, `240–260px` wide, scrollable if needed

- **Topbar:** Horizontal sticky header, shadow or border separation

- **Main content:** Flexible container with padding, responsive width

- **Grids:** Use auto-fit columns for cards/statistics panels

- **Spacing:** Consistent spacing units (e.g., `8px`, `12px`, `20px`, `32px`) for padding and gaps

## Panels & Cards

- **Shape:** Rounded corners (`6–12px`)

- **Background:** White or light gray

- **Border:** Subtle `1px` solid border

- **Shadow:** Soft elevation on hover or active

- **Hover effects:** Slight lift or border color change for interactivity

## Buttons

- **Shape:** Rounded corners (`6px`)

- **Primary buttons:** Strong color (blue) with white text

- **Secondary buttons:** Light background, subtle border, muted text

- **Hover/focus:** Slight color darkening or shadow increase

- **Padding:** Medium, comfortable click/tap targets

## Forms & Inputs

- **Inputs:** Rounded corners, subtle background, light border

- **Focus state:** Highlight border with primary color, slight shadow

- **Labels:** Uppercase or semi-bold for clarity

- **Layout:** Flexible grid or vertical spacing, consistent gaps

## Tables

- **Borders:** Subtle between rows and columns

- **Header row:** Uppercase, small font, muted color

- **Rows:** Hover effect with light background

- **Cells:** Padding for readability

## Badges / Status Indicators

- **Shape:** Rounded or pill-like

- **Color coding:**

    - **Success** → green

    - **Warning** → amber/yellow

    - **Error** → red

- **Background:** Soft pastel variant of color

- **Text:** Bold, readable

- **Optional:** Small dot or icon to reinforce meaning

## Search & Filters

- **Inputs**: Rounded, subtle background, inline icon on the left

- **Focus:** Highlight border, slight elevation

- **Placeholder:** Clear, descriptive, muted text color

## Design Principles

- **Consistency:** Spacing, colors, typography uniform across all modules

- **Clarity:** Minimal, readable text; visual hierarchy clear

- **Responsiveness:** Flexible grids and containers adapt to screen size

- **Interactive feedback:** Hover, focus, active states on all interactive elements

- **Enterprise feel:** Clean, professional, not cluttered; communicates value and reliability
